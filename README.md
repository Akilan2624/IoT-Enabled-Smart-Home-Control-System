# IoT-Enabled Smart Home Control System

## Overview
This project presents an IoT-based Smart Home Control System developed using ESP32. It enables users to remotely control household electrical appliances through a Wi-Fi connection. The system provides a simple, reliable, and cost-effective solution for smart home automation.

## Objectives
- Control home appliances remotely.
- Improve convenience and energy efficiency.
- Reduce manual operation.
- Build a low-cost smart home solution.

## Components Used
- ESP32 Development Board
- Relay Module
- Power Supply
- LED/Bulb/Fan
- Breadboard
- Jumper Wires

## Software Used
- Arduino IDE
- Embedded C++
- Wi-Fi

## Working
The ESP32 connects to a Wi-Fi network and hosts a simple web interface. When a user sends an ON or OFF command, the ESP32 controls the relay module, which switches the connected appliance accordingly.

## Features
- Wireless appliance control
- Real-time operation
- Low power consumption
- Easy to use
- Expandable for multiple devices
